# StudyFlow

Aplicativo educacional com foco em organização de estudos, gerenciamento de tarefas, quizzes e gamificação.

## Funcionalidades implementadas

- Planejamento de estudos com criação de tarefas por matéria, prioridade, prazo e observações.
- Lista de tarefas pendentes e concluídas com ganho de XP por conclusão.
- Sistema de gamificação com XP, nível, barra de progresso, conquistas e histórico de atividades.
- Quiz interativo com 4 alternativas, validação de resposta e recompensa em XP.
- Perfil do usuário com dados acadêmicos e pessoais.
- Assistente inteligente com lembretes, motivação e sugestão de rotina.
- Aba Explorar com notícias, dicas de estudo, habilidades do futuro e planejamento de carreira.
- Configurações de idioma, tema, notificações, conta e permissões.

## Rodando localmente

```bash
npm install
npm run dev
```

## Stack

- Frontend: React + TypeScript + Vite
- Arquitetura preparada para integração futura com backend Node.js/Firebase e Firestore/PostgreSQL.
